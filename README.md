# Sales Overview Dashboard

## Project Overview
End-to-end data analytics project using Python, PostgreSQL and Power BI.

## Tools Used
- Python (Pandas, SQLAlchemy)
- PostgreSQL
- Power BI Desktop

## Project Steps
1. Generated 5000-row sales dataset using Python
2. Loaded data into PostgreSQL using SQLAlchemy
3. Connected PostgreSQL to Power BI
4. Built interactive Sales Dashboard

## Dashboard Features
- KPI Cards: Total Sales, Total Profit, Total Quantity, Avg Rating
- Sales by Month (Line Chart)
- Sales by Region (Donut Chart)
- Sales by Category (Bar Chart)
- Sales by Product (Bar Chart)
- Sales by Payment Method (Pie Chart)
- Interactive Slicers: Category, Region
